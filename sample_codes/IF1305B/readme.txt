概要

アーカイブの内容
=================
-----------------------------------------------------
フォルダ構成
\tsujimi
├─PIC
├─HOGP
├─HOGP2010
└─LOG

-----------------------------------------------------
○PICフォルダ
　汎用I/OアダプタのPICに書き込むファームウェアです．

-----------------------------------------------------
以下のHOGPフォルダとHOGP2010フォルダはPC用のアプリケーション・ソフトウェアで，両者に本質的な違いは有りません．両者の違いは，前者がネイティブ・コードで，後者がマネージド・コードで書かれているということだけです．

○HOGPフォルダ
　PC用のアプリケーション・ソフトウェアです（ネイティブコード）．VC++6.0でコンパイルしていますが，Microsoft社のVisual Studioのダウンロード
Webページ（http://www.microsoft.com/visualstudio/jpn/）の［2012 評価版］からダウンロードできるVisual Studio Professional 2012（90日間試用版）を利用することもできます．

○HOGP2010フォルダ
　PC用のアプリケーション・ソフトウェアです（マネージドコード）．Visual C++/CLI(2010)でコンパイルしていますが，Microsoft社のVisual StudioのダウンロードWebページ（http://www.microsoft.com/visualstudio/jpn/）の［2012 評価版］からダウンロードできるVisual Studio Express 2012 for Windows 8を利用することもできます．

-----------------------------------------------------
○LOGフォルダ
汎用I/Oアダプタにデバック・モニタ回路をつないで，汎用I/OアダプタとPC間で送受信されたデータパケットをPC上でモニタした記録（ログ）ファイルです．
